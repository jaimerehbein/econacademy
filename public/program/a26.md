<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">Una variable aleatoria es una función que asigna un valor numérico a cada resultado de un experimento aleatorio. Intuitivamente, es cualquier característica que puede tomar diversos valores en un grupo de individuos <span class="hidden" data-ref="1" aria-hidden="true"></span>, <span class="hidden" data-ref="2" aria-hidden="true"></span>.</p>

Sus parámetros son $n$ (número de ensayos) y $p$ (probabilidad de éxito). Tiene un valor promedio y una varianza definidos <span class="hidden" data-ref="3" aria-hidden="true"></span>.</p>

Conceptos clave incluyen la población (totalidad de individuos) y la muestra (subconjunto representativo) <span class="hidden" data-ref="12" aria-hidden="true"></span>, <span class="hidden" data-ref="13" aria-hidden="true"></span>.</p>

</div>

<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">Dos variables cualitativas son independientes si la ocurrencia de una categoría en una variable no afecta la probabilidad de ocurrencia de las categorías de la otra.     <h3 class="text-xl font-bold text-emerald-300 tracking-tight">Frecuencias observadas y esperadas</h3>
</div>

<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">En una tabla de contingencia, se comparan los conteos reales (observados) con los que se esperarían si las variables fueran totalmente independientes (calculados como el producto de los totales marginales dividido por el total general) <span class="hidden" data-ref="37" aria-hidden="true"></span>.</p>
<!-- section: 26.11.3. -->
<div class="flex items-center gap-4 mt-10 mb-4">

    <h3 class="text-xl font-bold text-emerald-300 tracking-tight">Cálculo del contraste</h3>
</div>

<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">Se utiliza la estadística Chi-cuadrado de independencia. Se suman las diferencias entre frecuencias observadas y esperadas al cuadrado, divididas por las esperadas. Si el valor calculado excede el crítico, se rechaza la independencia <span class="hidden" data-ref="36" aria-hidden="true"></span>.</p>

<!-- section: 26.15.2. -->
<div class="flex items-center gap-4 mt-10 mb-4">
    <span class="text-lg bg-sky-500/15 border-sky-500/30 border w-9 h-9 rounded-xl flex items-center justify-center flex-shrink-0 shadow-sm">🔢</span>
    <h3 class="text-xl font-bold text-sky-300 tracking-tight">Tipos de errores y ajustes del modelo</h3>
</div>

<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">Es necesario evaluar el ajuste del modelo, identificando valores atípicos (outliers) y puntos de alto apalancamiento que puedan distorsionar las estimaciones. Se utilizan diagnósticos de residuos para validar las condiciones del modelo <span class="hidden" data-ref="26" aria-hidden="true"></span>.</p>
<!-- section: 26.15.3. -->
<div class="flex items-center gap-4 mt-10 mb-4">
    <span class="text-lg bg-sky-500/15 border-sky-500/30 border w-9 h-9 rounded-xl flex items-center justify-center flex-shrink-0 shadow-sm">🔢</span>
    <h3 class="text-xl font-bold text-sky-300 tracking-tight">Extensiones del modelo lineal</h3>
</div>

<p class="text-slate-300 text-base md:text-lg leading-relaxed my-4">El modelo lineal puede extenderse para incluir interacciones entre predictores (donde el efecto de una variable depende de otra) y regresión polinomial para modelar relaciones no lineales <span class="hidden" data-ref="26" aria-hidden="true"></span>.</p>
